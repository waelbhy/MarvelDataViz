Qu'est-ce que le graphe des intéractions ?


Ce graphe permet de visualiser les intéractions du personnage séléctionné. Quand vous avez validé votre choix de personnage, un graphe s'affiche.

Ce graphe met en avant tous les comics Marvel dans lesquels apparaît ce personnage, ainsi que tous les personnages qu'il rencontre dans ces comics.

Cette visualisation est intéressante car elle permet de voir les différents liens entre chaque personnage. On peut même parfois être surpris de voir un personnage peu 'connu' être relié à de nombreux personnages.

Une fois que le graphe affiché, vous pouvez cliquer sur un noeud (représenté par un point jaune) pour afficher le graphe d'un autre personnage.

Dr. Strange gardera l'arborescence de votre chemin en bas à gauche afin que vous puissiez retourner dans le passé sans problème !
